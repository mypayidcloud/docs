## What is PayID?

PayID is a backend-as-a-service platform for building **identity-first financial applications**. It handles the hard parts of user identity so you can focus on your product.

<CardGroup cols={3}>
  <Card title="Authentication" icon="lock" href="/guides/authentication/overview">
    Secure login with email/password, magic links, OTP, and Google OAuth. Supports both session and Bearer token auth.
  </Card>
  <Card title="OAuth 2.0 & OIDC" icon="shield-check" href="/guides/authentication/oauth-oidc">
    A fully compliant OAuth 2.0 authorization server with PKCE, refresh tokens, token introspection, and an OpenID Connect discovery endpoint.
  </Card>
  <Card title="KYC Verification" icon="id-card" href="/guides/platform/kyc">
    Identity verification via QoreID and Plaid. Supports BVN, NIN, passports, and international IDs with webhook callbacks.
  </Card>
</CardGroup>
