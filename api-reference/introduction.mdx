The PayID Platform API is organized around REST. It uses standard HTTP methods, returns JSON responses, and uses conventional HTTP status codes to indicate success or failure.

## Base URL

```
https://api.payid.cloud
```

## Authentication

All API requests require authentication. See the [Authentication Guide](/guides/authentication/overview) for details.

```bash
# Using a Bearer access token
Authorization: Bearer <access_token>

# Using an API Key
Authorization: ApiKey <api_key>
```
